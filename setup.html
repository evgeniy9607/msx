Как установить

  1. Откройте Lampa на телевизоре
  2. Перейдите в Настройки → Дополнения
  3. Нажмите "Добавить"
  4. Вставьте URL плагина (скопируйте выше)
  5. Повторите для каждого плагина
  6. Для TorrServer: Настройки → TorrServer → вставьте адрес
  7. Перезагрузите Lampa
